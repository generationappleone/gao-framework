# Contributing to GAO

We love contributions! Whether it's a bug fix, new feature, or documentation improvement, here's how you can help.

## Development Setup

1.  **Clone the repo**: `git clone https://github.com/generationappleone/gao-framework`
2.  **Install dependencies**: `pnpm install`
3.  **Build components**: `pnpm build`
4.  **Run tests**: `pnpm test`

## Coding Standards

- **TypeScript**: No `any`. Use strict mode.
- **SOLID**: Follow single responsibility and dependency inversion.
- **Security**: Never raw SQL. Always validate inputs.
- **Aesthetics**: UI contributions must feel premium and modern.

## Pull Request Process

1.  Create a feature branch from `main`.
2.  Ensure all tests pass and linting is clean.
3.  Follow [Conventional Commits](https://www.conventionalcommits.org/).
4.  Submit PR with a clear description of changes.
