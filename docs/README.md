## Core Philosophy

1.  **TypeScript First**: Everything is typed, always. No `any`.
2.  **No .env**: Configuration is code. Typed, validated, and immutable.
3.  **Secure by Default**: OWASP Top 10 mitigations built-in.
4.  **Multi-Platform**: One core, multiple targets (Web, Desktop, Mobile).
5.  **Developer Experience**: High-quality CLI and testing tools.
